How to Use

Curl this host at the following URL: 

http://devwebsimageprocessor201.webs.dev:8081/updateIP?siteNames=<site1>,<site2>...
Every site you send will be updated in the dev image processor hosts file with your current IP address, allowing image uploads to function correctly for that site.

Note, the application expects that the site name you provide is fully qualified, so make sure you are passing <sitename>.webs.dev ...
